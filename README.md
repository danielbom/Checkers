
## **Checkers game**

Basic checkers game write in C. Estimated time spend: 8 hours (not continuos). 

### **Compile and run**
> gcc game.c -o game && ./game

### **Compile and run [DEBUG mode]**
> gcc game.c -o game -D DEBUG && ./game

### **Observations***
> **!Warning!**: Files "game.v[*].c" can has errors.
